# Learning Claude Code

Welcome to your Claude Code tutorial project!

## What is Claude Code?

Claude Code is an AI-powered coding assistant that can help you with various programming tasks directly from the command line.

## What Can I Do?

Here are some things you can try:

### 1. **File Operations**
- Create, read, edit, and write files
- Search for files and content across your project
- Understand existing code

### 2. **Coding Tasks**
- Write new functions or features
- Debug and fix errors
- Refactor code
- Add tests
- Explain how code works

### 3. **Git Operations**
- Create commits with well-formatted messages
- Create pull requests
- Check status and diffs

### 4. **Running Commands**
- Install dependencies
- Run tests
- Execute build scripts
- Start development servers

### 5. **Project Setup**
- Initialize new projects (Node.js, Python, etc.)
- Set up configuration files
- Create project structures

## Try These Examples

1. **"Create a simple Python script that prints 'Hello World'"**
2. **"Initialize a Node.js project with a basic Express server"**
3. **"Write a function that calculates Fibonacci numbers"**
4. **"Explain how [some code] works"**
5. **"Add error handling to [some function]"**

## Tips

- Be specific about what you want
- I can work on multiple files at once
- I'll read code before modifying it to understand context
- I can search through large codebases efficiently

Ready to start? Just tell me what you'd like to build or learn!
